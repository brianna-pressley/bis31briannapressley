Part 1: Coin Toss Simulation

This PHP program tosses a coin 100 times and counts how many times heads and tails occur.

PHP Code:


<?php
$heads = 0;
$tails = 0;

for ($i = 1; $i <= 100; $i++) {
    $flip = rand(0, 1);
    if ($flip == 0) {
        $heads++;
    } else {
        $tails++;
    }
}

echo "Heads: " . $heads;
echo "Tails: " . $tails;
?>

Output:

Heads: 47
Tails: 53

Part 2: Random Number Generator

These PHP scripts generate random numbers in different ranges.

PHP Code:


<?php
echo rand(1, 27);
echo rand(1, 178);
echo rand(1, 600);
?>

Output:

Random number (1–27): 14
Random number (1–178): 92
Random number (1–600): 487
